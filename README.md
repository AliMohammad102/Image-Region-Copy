# Image Region Copy

## This project was implemented as a part of the Multimedia course assignment.

This project implements a simple tool that copy a selected region of an image to a new position inside the same image.

It loads an image, extracts a patch using given coordinates, and pastes it into a new location.

## Files
- main.py - main code for noise generation and removal.
- FirstImage.jpg - original input image.
- NewFirstImage.jpg - output after copy a region from FirstImage.jpg and paste it in a new position in the same image.
- SecondImage.jpg,ThirdImage.jpg,NewSecondImage.jpg,NewThirdImage.jpg - Those are another examples about input images and output images. 

## How to Run
- python main.py
